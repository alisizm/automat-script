## 🇬🇧 English Description

### 📖 Overview
Add a realistic and interactive vending machine system to your server! Players can easily buy food and drinks from vending machines and drink water directly from water dispensers located around the map. It features a modern, clean NUI and is highly optimized for performance.

### ✨ Features
*   **Interactive Machines:** Buy food & drinks from vending machines scattered across the map.
*   **Water Dispensers:** Drink water for free from water dispensers.
*   **Modern UI:** Clean, optimized, and responsive NUI interface.
*   **Multi-language Support:** Built-in JSON-based locale system for easy translation.
*   **Highly Customizable:** Fully configurable via the config file to suit your server's needs.
*   **Framework Independent:** Easy to integrate into any server setup.
*   **Secure:** Robust server-side checks and anti-exploit protection.
*   **High Performance:** Optimized code for extremely low resource usage.

### ⚙️ Configuration
*   All core settings can be easily modified via `config.lua`.
*   Language files can be edited in `locales/*.json`.
*   The UI is fully customizable to match your server's aesthetic.

### 📦 Dependencies
*   **Required:** None (Standalone).
*   **Optional:** `ox_lib`, `ox_target`, `ox_inventory`.

### 🔐 Licensing & Protection
This resource is protected with the FiveM Asset Escrow system via Tebex. Client and server codes are securely encrypted to prevent unauthorized use and leaking.
